# Tesheng's Tour of Heroes

---

## 專案背景

---

本專案旨在練習 Angular v16 的相關概念，並建立一個英雄清單，可以提供使用者新增、修改、刪除、搜尋英雄。

## 使用技術

---

- Angular v16 Component, Directive, Pipe, Data Binding, Reactive Forms 與 Routing
- In-memory-api 模擬伺服器
- RxJs 處理API

## 本地端執行

---

git clone repo連結 cd 專案資料夾 在 terminal 輸入

```
npm start
```
